# FP&A Variance Analyzer

Automates the monthly budget vs. actuals variance review: raw CSV exports in, formatted Excel report, charts, and executive summary out.

Status: work in progress.  Built as a portfolio project to demonstrate finance process automation.  Full documentation and sample output coming as the build progresses.

Planned pieces:

- Sample budget and actuals data shaped like a standard ERP export
- Python script that calculates variances, applies favorability logic, and flags material items
- Auto-generated Excel report, charts, and executvie summary
